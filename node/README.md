# WhatTimeIsIn GeoIP (Node + TypeScript)

Free, local GeoIP API using Node.js and SQLite.

---

## Requirements

- Node.js 18+

---

## Run

From the language folder:

```bash
chmod +x ./start.sh
./start.sh
```

The API will be available at:

```
http://localhost:5022
```

---

## Database download

Download the database from:

```
https://whattimeis.in/public/downloads/
```

Save it into the project root at:

```
config/database/WhatTimeIsIn-geoip.db
```
